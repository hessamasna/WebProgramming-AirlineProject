## How to start

First, clone the repo:

```
git clone https://github.com/hessamasna/WebProgramming-AirlineProject
```

Then use npm to install all Dependences and run Project

```
npm i
npm css
npm start
```



> npm css = for compile Tailwind
> npm start = for run live server
